# How to Set Chrome as Default Browser on Windows 11

This guide explains how to change Windows 11â€™s default browser from Microsoft Edge to Google Chrome, including workarounds for system-enforced Edge behaviors.

> [!NOTE]
> The following steps require a third-party tool that may stop working after major Windows updates.

> [!WARNING]
> This was written in 2022. Instructions and information might be outdated.

## Changing Your Default Browser to Chrome

1. Click the **Start** button on your taskbar.

![Windows Start button highlighted in green on the task bar.](images/chrome-default-browser/step1.png)

2. Search for **Default apps**.

![Windows Start Menu Search Bar with 'default apps' written and highlighted in green.](images/chrome-default-browser/step2.png)

3. Click on the best match called **Default apps** from **System settings**.

![Windows Start Menu Best Match with 'Default apps' highlighted in green.](images/chrome-default-browser/step3.png)

4. Click on **Google Chrome** towards the bottom of the list.

![Windows Default Apps with Google Chrome highlighted in green.](images/chrome-default-browser/step4.png)

5. Click on **HTTPS** towards the bottom of the list.

![Windows Default Apps Google Chrome page with 'HTTPS Microsoft Edge' highlighted in green.](images/chrome-default-browser/step5.png)

6. Select **Google Chrome** from the list.

![Windows Menu 'How do you want to open this?' with 'Google Chrome' highlighted in blue.](images/chrome-default-browser/step6.png)

7. Click **OK** to change the default browser.

![Windows Menu 'How do you want to open this?' with the 'OK' button highlighted in green.](images/chrome-default-browser/step7.png)

8. Repeat steps 5, 6, and 7 for all necessary file associations.

> [!TIP]
> It's recommended to change the default settings for **.htm**, **.html**, and **.pdf** extensions.

## Changing Your Default Browser to Chrome for Widgets and Search

Due to Windows 11 design restrictions, changing the default browser for widgets and search requires a workaround.

To change the default browser for widgets, a free external software is required.

> [!NOTE]
> The following steps require a third-party tool that may stop working after major Windows updates. 

1. Download [EdgeDeflector](https://github.com/da2x/EdgeDeflector/releases).

![ website with the software installer highlighted in green.](images/chrome-default-browser/step1-1.png)

2. Open the file.

![Windows context menu for downloaded file with the 'Open' option highlighted in green.](images/chrome-default-browser/step1-2.png)

3. Click the **Start** button on your taskbar.

![Windows Start button highlighted in green on the task bar.](images/chrome-default-browser/step1-3.png)

4. Search for **Default apps**.

![Windows Start Menu Search Bar with 'default apps' written and highlighted in green.](images/chrome-default-browser/step1-4.png)

5. Click on the best match called **Default apps** from **System settings**.

![Windows Start Menu Best Match with 'Default apps' highlighted in green.](images/chrome-default-browser/step1-5.png)

6. Click on **EdgeDeflector** towards the bottom of the list.

![Windows 'Default apps' page with EdgeDeflector highlighted in green.](images/chrome-default-browser/step1-6.png)

7. Click on **Microsoft Edge**.

![Windows 'Default apps' page with 'MICROSOFT-EDGE' highlighted in green.](images/chrome-default-browser/step1-7.png)

8. Select **EdgeDeflector** from the list.

![Windows Menu 'How do you want to open this?' with 'EdgeDeflector' highlighted in blue.](images/chrome-default-browser/step1-8.png)

9.  Click **OK** to confirm the change.

![Windows Menu 'How do you want to open this?' with the 'OK' button highlighted in green.](images/chrome-default-browser/step1-9.png)

If EdgeDeflector stops working after Windows 11 updates, check the software's website for new updates.
